## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/Vasu7777py/myterminal

# Access
$ cd myterminal

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
